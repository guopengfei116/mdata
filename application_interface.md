# 接口文档

## 一、约定

### 1、服务端数据返回格式

>####1.  整体格式：
>> { msg : 错误描述， code : 约定的状态码， data : 返回数据 }

>####2.  状态码约定：
>>*  200 : 操作成功  
>>*  401 : 操作失败
>>*  404 : 不存在
>>*  405 : 权限不足
>>*  500 : 服务器错误

>####3.  错误描述：
>>* 当code != 200时，取msg作为参考错误提示，需要时展示给用户
